# how to play
- ```python3 main.py```
- Avoid asteroids to survive
- __W S__ forward and backward movement
- __A D__ rotate clockwise and counterclockwise
- __SPACE__ Shoot a bullet
- __X__ or __Alt+f4__ to quit
 have fun
